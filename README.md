# FakeJayShop
- 안드로이드 권장 아키텍쳐 연습 프로젝트 입니다.

## 아직 작업 중 입니다.

## 이 프로젝트는 https://fakestoreapi.com/ 를 활용하여 쇼핑몰 앱을 만드는 프로젝트 입니다.

## 기술 스택
- 앱 아키텍쳐 
  - Clean Architecture 기반
  - https://developer.android.com/topic/architecture?hl=ko
- core 폴더 기반 모듈화
- Networking - Retrofit
- DI - hilt
- UI - Compose First
- localData - Room, DataStore
